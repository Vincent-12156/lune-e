# LUN-E

## PREAMBULE

J'ai créé cette entreprise dans le milieu du développement web et web-mobile pour aider pour des particuliers et des TPE qui ont besoins de solutions personnalisées tant au niveau visibilité (site internet) que d'applications de gestion.

Je propose actuellement une app mobile gratuite pour l'échange de biens dans le domaine du jardinage, une autre pour éditer des partions de musique et un projet de portfolio pour une artiste peintre.

Mon projet du moment est une application de gestion pour TPE qui fait gagner du temps par son interface rapide ; permettant de créer un fichier clients, éditer des factures et des compte-rendus de prestations avec une visibilité de l'activité grâce à son tableau de bord et donc un rapprochement comptable.
